# Next App Template

## Usage

```bash
# Init the project
bunx create-next-app --example https://github.com/r-portas/next-app-template

# cd into the app
cd <my-app>

# Rename the workspace
mv next-app-template.code-workspace <my-app>.code-workspace
```

## Tooling

- [Bun](https://bun.sh)
- [Next.js](https://nextjs.org)
- [Biome](https://biomejs.dev)
